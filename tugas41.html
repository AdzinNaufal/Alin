Determinan Matriks#

Pengertian Determinan Matriks#

Nilai determinan matriks memberikan informasi tentang apakah matriks tersebut memiliki invers (jika determinannya tidak nol), apakah matriks tersebut singuler atau tidak. Jika determinan suatu matriks adalah nol, maka matriks tersebut disebut sebagai matriks singular, yang artinya tidak memiliki invers. Sedangkan jika determinannya bukan nol, maka matriks tersebut disebut sebagai matriks nonsingular atau invertible, yang artinya memiliki invers. Mencari determinan matriks hanya dapat dilakukan pada matriks bujur sangkar atau matriks persegi atau matriks yang mempunyai baris = kolom

Sifat-Sifat Determinan Matriks#

Membuat matrix identitas dengan code fungsi python untuk membuktikan determinan matrix identitas (sifat-sifat determinan).

Sifat-sifat Matrix determinan:

  1. Jika baris Matrix Identitas ditukar maka akan mengubah tanda dari Determinan Matrix Identitas tersebut

  2. Jika di dalam suatu Matrix ada baris atau yang merupakan kelipatan dari baris atau kolom yang lain maka det = 0.

  3. Jika ada Matrix kelipatan dari matrix lain maka determinan matrix tersebut adalah kelipatan pangkat ordo kali determinan matrix dari kelipatannya.

  4. Jika Determinan Matrix B adalah 2x Determinan Matrix A maka ada salah satu baris Matrix B yang menjadi 2x lipat baris Matrix A.

  5. Jika ada matrix yang berasal dari penjumlahan matrix A dan matrix B maka determinannya adalah penjumlahan dari matrix A dan Matrix B.

  6. Jika ada Matrix segitiga atas maka determinannya adalah perkalian anggota diagonal Matrix tersebut.

COFACTOR MATRIKS#

Menghitung Invers Matriks 5 x 5 Dengan Kofaktor#

Berikut saya sajikan matriks S dengan ordo 5 x 5:

\begin{align*} S= \begin{bmatrix} 4&2&3&1&3\ 2&2&1&3&2\ 4&1&3&2&1\ 1&3&1&1&2\ 2&4&1&1&3 \end{bmatrix} \end{align*}

Invers dari matriks di atas akan dicari dengan cara mencari determinan dan kofaktor - kofaktornya terlebih dahulu yang akan diimplementasikan melalui code. Sebelum itu, berikut saya sajikan rumus - rumus untuk menyelesaikan persoalannya:

  • Rumus Kofaktor : \(C_{ij} = (-1)^{i+j} * detS\)

  • Rumus Determinan(Dengan metode pick row atau pick coulumn) : \(a_{ij} * c_{ij} + a_{ij} * c_{ij} + a_{ij} * c_{ij} + a_{nn} * c_{nn}\)

  • Rumus invers matriks : \({ \frac{1}{detS}} * adj(S)\)

  • Rumus mencari Adjoin : \(adj(S) = (kof(A))^T\)

Adjoin (adj(s)) merupakan transpose dari setiap kofaktor matriks
